This volume describes the dataset that is exchanged.

The PZP dataset reuses the work and specifications of IKNL. IKNL has done substantial work on the
PZP dataset, and this guide reuses it rather than redefining it. See the IKNL PZP R4 documentation:

- <https://as-iknl-api-documentatie.azurewebsites.net/docs/pzp/r4/en/index.html>
